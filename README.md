# AI Resume Builder

A modern, beautiful AI-powered resume and CV builder with multiple templates and color schemes!

## Features ✨

- **User-friendly Form**: Fill in your personal, professional, and educational details
- **Profile Image Upload**: Add your profile picture
- **Multiple Templates**: Choose from 5 professional templates
  - Modern
  - Classic
  - Creative
  - Minimalist AI
  - Elegant Executive
- **10 Premium Color Schemes**: Beautiful gradient color combinations
- **Resume & CV Modes**: Toggle between resume and CV views
- **PDF Download**: Download your resume as a PDF file
- **Responsive Design**: Works on all screen sizes

## Tech Stack 🛠️

- **Frontend**: React + Vite
- **Backend**: Node.js + Express
- **Styling**: Custom CSS

## Installation & Usage 🚀

### Prerequisites
- Node.js installed on your computer

### Steps:
1. Clone or download this repository
2. Install dependencies for both frontend and backend:
   ```bash
   cd frontend
   npm install
   cd ../backend
   npm install
   ```
3. Start the backend server:
   ```bash
   cd backend
   npm start
   ```
4. Start the frontend:
   ```bash
   cd frontend
   npm run dev
   ```
5. Open your browser and navigate to http://localhost:3000
